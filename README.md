# ❄️ Hokkaido Weather Dashboard for Notion

홋카이도 주요 도시(삿포로, 아사히카와, 하코다테, 쿠시로)의 5일간 일기예보를 한눈에 확인할 수 있는 노션 전용 날씨 위젯입니다. 
노션의 라이트/다크 모드에 완벽하게 대응하며, Phosphor Icons를 사용해 세련된 대시보드 UI를 제공합니다.

## ✨ Key Features

- **Hokkaido 4 Cities**: 삿포로, 아사히카와, 하코다테, 쿠시로의 실시간 기상 데이터 연동
- **5-Day Forecast**: 오늘을 포함한 5일간의 최저/최고 기온 및 날씨 상태 표시
- **Notion Optimized**: 노션 임베드 환경에 최적화된 콤팩트한 레이아웃 및 해상도
- **Auto Dark Mode**: 사용자 시스템 설정 및 노션 테마에 따라 다크 모드 자동 전환
- **Premium UI**: Phosphor Icons 기반의 벡터 아이콘과 세련된 타이포그래피 적용
